Build a complete static React.js web admin panel called
"WorkSetu Admin Panel" — for a location-based service booking
and workforce dispatch platform. This is the SUPER ADMIN
side only. Single admin role with full access to everything.

==============================================
TECH & GENERAL RULES
==============================================

- Framework: React.js
- UI Library: Ant Design (antd)
- All data is hardcoded — NO backend, NO API calls
- Use React Router for all navigation
- Every screen must be a separate component file
- Use React useState and useEffect hooks for local state
- Charts: use Recharts library
- Icons: Ant Design icons only
- Fully responsive web layout
